# 校园失物招领前端

## 运行方式

```bash
npm install
npm run dev
```

默认开发地址：

- 前端：`http://localhost:5173`
- 后端代理：`http://localhost:8080`

## 已接入功能

- 登录、注册
- 失物/拾物列表查询
- 发布、编辑、删除、找回/认领
- 评论、举报、私信、通知
- 置顶申请
- AI 描述生成与重生成
- 个人中心
- 管理员用户管理、内容删除、统计、举报/置顶审批、AI 总结

## 注意事项

- 管理员页面需要后端返回的 `role=1` 账号登录。
- Vite 已配置 `/api` 代理到本地 `8080` 后端。
- 时间字段按后端要求使用 `yyyy-MM-dd HH:mm:ss`。
