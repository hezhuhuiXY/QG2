## 表单标签

| 标签         | 描述                                   |
| ------------ | -------------------------------------- |
| `<form>`     | 定义表单                               |
| `<input>`    | 定义表单项，通过 type 属性控制输入形式 |

**form：定义表单**

- **action**：规定当提交表单时向何处发送表单数据，URL
- method：规定用于发送表单数据的方式
  - `get`：浏览器会将数据直接附在表单的 action URL 之后。大小有限制
  - `post`：浏览器会将数据放到 http 请求消息体中。大小无限制

| `reset`    | 定义重置按钮，重置按钮会清除表单中的所有数据   | 「重置」按钮                       |

## CSS

css导入HTML方式

```bash
1. <div style="color: red">Hello CSS~</div>

2.<style type="text/css">
    div{
        color: red;
    }
</style>

3.<link rel="stylesheet" href="demo.css">
```

| 选择器类型 | 语法格式                    | 说明                                                         |
| ---------- | --------------------------- | ------------------------------------------------------------ |
| 元素选择器 | `元素名称{color: red;}`     | 直接通过 HTML 标签名选中所有对应元素，作用于页面中所有该标签 |
| id 选择器  | `#id属性值{color: red;}`    | 通过元素的`id`属性选中**唯一**对应元素（id 在页面中不可重复） |
| 类选择器   | `.class属性值{color: red;}` | 通过元素的`class`属性选中对应元素，可重复使用，支持一个元素绑定多个类 |

## 正则表达式

| 符号 / 表达式 | 含义                     | 等价 / 说明                                 |
| ------------- | ------------------------ | ------------------------------------------- |
| **^**         | 匹配字符串的**开始**位置 | 必须以后续内容开头                          |
| **$**         | 匹配字符串的**结束**位置 | 必须以前置内容结尾                          |
| **[]**        | 匹配范围内的**单个字符** | 如 `[0-9]` 匹配单个数字                     |
| **.**         | 匹配**任意单个字符**     | 不含换行符                                  |
| **\w**        | 匹配**单词字符**         | 等价于 `[A-Za-z0-9_]`（字母、数字、下划线） |
| **\d**        | 匹配**数字字符**         | 等价于 `[0-9]`                              |

| 符号      | 含义                         |
| --------- | ---------------------------- |
| **+**     | 至少出现 **1 次**            |
| *****     | 出现 **0 次或多次**          |
| **?**     | 出现 **0 次或 1 次**         |
| **{x}**   | 恰好出现 **x 次**            |
| **{m,}**  | 至少出现 **m 次**            |
| **{m,n}** | 至少 **m 次**，至多 **n 次** |
